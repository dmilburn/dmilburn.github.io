What does it mean to display “inline” vs. “inline-block”?

1/21/2015

HTML and CSS are programming languages used in designing websites. HTML is what you use to display the content of the website and CSS is what you use to change the formatting, color and everything involved in making websites pretty.

One element involved in CSS formatting is changing the "display" property. Display effects how an object is displayed on the screen. Here is the correct CSS syntax for using “display”:

.awesome_possum {

display: block;

}

In the above code, you can see the word "block" after "display". “Block” is one of many different display properties that one can use in designing websites. Today I will explain the differences between "block", “inline” and “inline-block” displays.

Inline-Block Objects

As the name suggests, objects defined as inline-block have features similar to both inline objects and block objects. Like blocks, inline-blocks can also have "physical" attributes; you can set the height and width of inline blocks. Like inline objects, objects that are inline-blocks move to be in a horizontal line with each other. Also similarly to inline objects, you cannot adjust the right and left margins of inline-blocks, only the top and bottom margins; therefore, you can only directly move inline-blocks up and down, not left and right.
