How the Internet works

3/8/15

The Internet is a series of connected computers that communicate and exchange data. In order to keep track of all of the different possible computers, the Internet Protocol (IP) system was introduced. Every computer has its own IP address which is a series of 4 to 12 numeric digits.

Every website can be accessed using its IP address. However, since humans are better at remembering words than numbers, a system was created to make the websites more memorable. For example, when you type www.google.com into your browser, you are really entering its IP address, 216.58.217.4. Domain names are converted to IP addresses by use of the Domain Name Service (DNS).

Once a domain name is converted to its IP address, the server attached that that IP address is contacted with a Hypertext Transfer Protocol (HTTP) request. If the request is invalid, a 404 error is returned. If the request is valid, the browser receives the information and displays it.
