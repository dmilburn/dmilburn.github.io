JavaScript Array Method Cheat Sheet

Method or Property:

Example:

Find the length of an array.

ary.length;

var ary = [3, 2, 4, 1];

ary.length;

=> 4

Search an array for something and have its index returned

ary.indexOf();

var ary = [3, 2, 4, 1];

ary.indexOf(3);

=> 0

Iterate over an array.

for loop

var numbers = [1, 2, 3, 4, 5];

for (var i = 0; i < numbers.length; i++)

{ numbers[i] *= 2; }

// numbers is now [2, 4, 6, 8, 10]

Return the elements in an array that satisfy the conditions of a function

ary.filter(nameOfFunction());

function isBigEnough(element) {

return element >= 10; }

var filtered = [12, 5, 8, 130, 44].filter(isBigEnough);

=> filtered is [12, 130, 44]

Sort an array of Strings in place according to alphanumeric value.

ary.sort();

var ary = ["cat", "apple", "zebra", "horse"];

ary.sort();

=> ["apple", "cat", "horse", "zebra"]

Sort an array of numbers.

*ascending*ary.sort(function(a, b){return a-b});

*descending*ary.sort(function(a, b){return b-a});

var points = [40, 100, 1, 5, 25, 10];

points.sort(function(a, b){return a-b});

=>[1, 5, 10, 25, 40, 100]

var total = [0, 1, 2, 3].reduce(function(a, b) {

return a + b;

=> total == 6
